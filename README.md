# heroku react boilerpate project

## a full-stack app with ES6/ES2015 React.js featuring universal Redux, CSS modules, Express 4.x

#Npm Scripts
Install dependencies
```
npm install
```
Run dev server
```
npm run start-dev
```
Run Linting
```
npm run lint
```
Run Test
```
npm run test-ci
```
Production build(prodution bundles are in /public )
```
npm run build
```
